# odin-recipes
project 1-TOP-html

Introduction

This project forms part of the Foundations course from The Odin Project, specifically the Recipes assignment. In this exercise, I built a simple multi-page website using only HTML to present a collection of recipes. The focus was not on visual styling, but on structuring content clearly and correctly, linking pages together, and creating a functional, navigable site.

Through completing this project, I practiced the core building blocks of web development. I created structured HTML documents using headings, paragraphs, images, and both ordered and unordered lists. I also linked multiple pages together through a central index page, reinforcing how websites are organized at a fundamental level. In addition, I used Git to manage my work and became more comfortable with a basic development workflow.

By the end of this project, I have a stronger grasp of how raw HTML shapes the structure of a webpage, and I feel more confident navigating the tools and processes that underpin modern web development.
